# gmtk2025

### Playing the game

Run `lime run html5`

### Controls

**Flippers:**
- **Z** - Left flipper
- **M** - Right flipper

**Game:**
- **P** or **ESC** - Pause/unpause game
- **R** - Restart current level

**Debug:**
- **Arrow Keys** - Move ball directly (debug mode)
- **A** - Invert ball color (debug mode)
